# Exercise 3: Formatted Printing

## 学习目标

- 使用printf进行格式化输出
- 理解格式占位符和转义序列
- 理解宽度和精度
- 观察缺少参数和未初始化变量的警告
- 修改Makefile同时构建ex1和ex3

## printf基本形式

    printf("格式字符串", 参数...);

例如：

    printf("I am %d years old.\n", age);

`%d`表示一个`int`类型整数，`\n`表示换行。

## 常用格式占位符

- `%d`：有符号整数
- `%u`：无符号整数
- `%f`：浮点数
- `%c`：字符
- `%s`：字符串
- `%x`：十六进制
- `%p`：指针
- `%%`：百分号

## 常用转义序列

- `\n`：换行
- `\t`：Tab
- `\r`：回到当前行开头
- `\\`：反斜杠
- `\"`：双引号

## 宽度与精度

    printf("%5d", 42);
    printf("%05d", 42);
    printf("%.2f", 3.14159);

## 缺少参数实验

格式字符串中包含`%d`，但没有提供对应参数时，编译器会警告格式参数不足。

程序继续运行会产生未定义行为，可能输出随机值或崩溃。

## 未初始化变量实验

    int age;

局部变量没有初始化，其值是不确定的。读取它属于未定义行为。

## Makefile

    CFLAGS=-Wall -Wextra -g

    all: ex1 ex3

    clean:
    	rm -f ex1 ex3

    .PHONY: all clean

执行：

    make clean all

会删除旧程序并重新构建`ex1`和`ex3`。

## printf返回值

`printf`成功时返回实际输出的字符数，失败时返回负数。

## 本题总结

格式字符串必须与参数类型和数量严格对应。编译器警告不能忽视，因为错误的格式参数和未初始化变量都可能导致未定义行为。
